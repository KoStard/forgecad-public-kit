# Task: Mounting Bracket with Bolt Holes

Build a single-piece L-shaped mounting bracket with bolt holes.

## Requirements

Return a single **Shape** (not an assembly).

The bracket must:
- Be L-shaped (two perpendicular flat plates joined at a right angle)
- Have **two through-holes**, one in each plate, sized for M6 bolts (6.5mm diameter)
- Wall thickness: 5mm
- Each plate: approximately 40mm x 30mm
- The holes must go completely through the plate (not blind holes)

## Key dimensions
- Plate size: ~40mm x 30mm each
- Plate thickness: 5mm
- Hole diameter: 6.5mm (M6 clearance)
- Holes centered in each plate

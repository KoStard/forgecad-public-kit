# Task: 90-Degree Pipe Elbow

Build a single-piece 90-degree pipe elbow fitting.

## Requirements

Return a single **Shape** (not an assembly).

The elbow must:
- Turn 90 degrees (one opening faces +X, the other faces +Z, or similar perpendicular pair)
- Be **hollow** — a tube, not a solid bend
- Have an outer diameter of 30mm and a wall thickness of 3mm (inner diameter 24mm)
- Both openings must be clear (a rod can pass through either end)
- The bend radius (centerline) should be approximately 40mm

## Key dimensions
- Outer diameter: 30mm
- Wall thickness: 3mm
- Inner diameter: 24mm
- Bend centerline radius: ~40mm

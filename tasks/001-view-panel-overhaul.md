# View Panel Overhaul

Make the view panel a proper control center for the 3D viewport.

## Current State
Basic viewport with minimal controls.

## Requirements

### Render Modes
- Wireframe toggle
- Solid / shaded rendering
- Wireframe + solid overlay mode

### Object Visibility
- When multiple objects are imported/created, list them in a panel
- Per-object show/hide toggle
- Per-object opacity/transparency control

### View Controls
- Orthographic / perspective toggle
- Snap to standard views (front, back, top, bottom, left, right, iso)
- Fit-to-view / zoom-to-selection

### Nice to Have
- Object color override per object
- Section clipping plane (interactive)
- Grid toggle and grid size control

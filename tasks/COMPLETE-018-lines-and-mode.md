# Phase 1
I should be able to create lines, shapes, polygons, from dots/edges that I extract from other objects.
For example, if I have a circle, I should be able to say "give me the middle point of it", then get another one from somewhere else, and create a line.
Similarly, I should be able to create a polygon. And should be able to "close" the polygon too.
Also, intersection point of two lines should be easy to get.
Should be able to do something like this:
- given 2 rectangles
- for each rectangle, get the 2 diagonals, get the intersection point
- create a line between these two dots

for polygons, the naming of the edges should be clockwise.
